# Caramel

Caramel は Pulsate の軽量化クライアントで, 最低限の機能のみが実装されています.

- アカウントへのログイン・ログアウト
- タイムラインの取得・表示
- テキスト投稿
- 制限されたリアクション

> [!IMPORTANT]
> 
> Caramel は検証などの限られた用途でのみ使用することをお勧めします. 本番環境での使用は推奨されません.

> [!NOTE]
> 
> 現在 Caramel は開発中です. いくつかの機能が未実装であるか, または不安定である可能性があります.
